This directory contains key statutory and regulatory text excerpts
used by the compliance lattice in `compliance.py`. Each file contains
the operative provisions that the lattice evaluates.

| `hipaa.md` | HIPAA Privacy & Security Rules | 45 CFR §164.530, §164.502 |
